# 解題思路

### 目的：
找到輸入字符串 s 中的最長回文子串。回文子串是指從前往後讀和從後往前讀都相同的子串。
### 方法：
- 使用 "中心擴展法" 來解決這個問題。這個方法的核心思想是：

1. 對於字符串中的每個字符，我們認為它是回文的中心。
2. 我們從中心向外擴展，直到遇到不匹配的字符，記錄此回文的長度。
3. 對於每個字符，我們考慮兩種情況：
    - 該字符自身作為中心（對應奇數長度的回文）。
    - 該字符與其下一個字符共同作為中心（對應偶數長度的回文）。
