# 解題思路

### 目的：
將羅馬數字轉換成一般的阿拉伯數字。
### 方法：
- 使用 HashMap 將羅馬數字字符對應到其整數值，例如 'I' 對應 1，'V' 對應 5，依此類推。
- 設置一個整數變量 result，初始值為 0，用於累加最終的整數值。
- 使用 for 迴圈從字符串的第一個字符遍歷到最後一個字符。current 變量保存當前字符對應的整數值。
- 如果當前字符不是字符串中的最後一個字符，且當前字符的值小於下一個字符的值（例如 IV 中的 I 小於 V），則需要從 result 中減去當前值。否則，將當前值加到 result 中。
- 最後返回 result，它即為轉換後的整數值。

Java
